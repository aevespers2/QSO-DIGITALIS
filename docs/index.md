## Decision in plain language

The portfolio needs to decide whether a separate repository is useful for describing how inert source references become separately identified interpretations, purpose-limited projections, and exchange envelopes. A valid charter would define only its accepted profile and governance. It would not own the source evidence, temporal truth, transport, independent disposition, consumer runtime, experiment conclusions, interface approval, or operational authority.

The decision must be explicit. A repository name, documentation quality, mergeability, inactivity, preserved historical design, or successful checks cannot approve a role. When a unique role is not demonstrated, the safe outcome is a provenance-preserving split, migration, or retirement rather than inferred implementation scope.

- literal consciousness or autonomous authority.
